    Directorios con el formato: Actividades <número> 
        donde <número> es un entero entre 1 y 5.
        
    
    Contiene los resultados de las actividades que se encuentran descritas
    en el archivo "Proyecto.txt" y archivos necesarios para generar 
    dichos resultados, como: algoritmos de búsqueda, heurísticas, 
    abstracciones, etc. 
    
--------------------------------------------------------------------------------
    Directorio Problemas
    
    Contiene los archivos .psvn necesarios para generar los resultados
    solicitados en las actividades descritas en el archivo "Proyecto.txt";
    así como los generadores usados para obtener dichos archivos.
